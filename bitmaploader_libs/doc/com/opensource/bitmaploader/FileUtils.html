      • parseUriToFile

        public static java.io.File parseUriToFile(android.content.Context context,
                                          android.net.Uri uri)
        Parse a content uri to a file. Some file manager return Uri like "file:///sdcard/test.mp4", In this case Uri.getPath() get the file path in file system, so can create a file object with this path, if this file is exists, means parse file success. Some file manager such as Gallery, return Uri like "content://video/8323", In this case Uri.getPath() can't get file path in file system, but can user ContentResolver to get file path from media database.
        参数:
        uri -
